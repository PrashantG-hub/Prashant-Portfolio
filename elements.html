Flight Price Prediction EDA

End-to-end EDA of flight pricing to uncover key price drivers by data cleaning, feature engineering, and visual analysis

The Flight Price Prediction EDA notebook focuses on understanding the key factors that influence flight ticket prices using Python. The analysis starts by loading the dataset with pandas and getting familiar with its structure, such as the number of records, features, data types, and missing values. A major part of the work involves cleaning and preparing the data. Date and time columns are converted into proper datetime formats, useful components like day, month, hour, and minute are extracted, and missing or duplicate values are handled carefully. Flight duration is also transformed into numerical values, making the data easier to analyze and more meaningful for further exploration. Once the data is cleaned, exploratory analysis is carried out using descriptive statistics and visualizations. Charts like histograms, box plots, and bar plots are used to understand price distribution, detect outliers, and analyze how factors such as airline, source, and destination affect ticket prices. Overall, this notebook demonstrates a clear, structured EDA workflow and highlights how thoughtful data preparation and visualization can reveal real-world pricing patterns, setting a strong foundation for predictive modeling.

    Google Play Store EDA

    End-to-end EDA of app data to understand user behavior and app performance through data cleaning, feature analysis, and visualization

    The Google Play Store EDA notebook focuses on analyzing mobile application data to identify patterns that influence app performance and user engagement. The analysis begins by loading the dataset using pandas and understanding its structure, including the number of apps, available features, data types, and missing values across columns. A significant part of the analysis involves data cleaning and preprocessing. This includes handling missing values, removing duplicate app entries, standardizing categorical fields such as app categories and content ratings, and converting columns like app size, installs, and price into appropriate numerical formats. Ratings and review counts are also validated to ensure consistency and reliability for analysis. Once the data is cleaned, exploratory analysis is performed using descriptive statistics and visualizations. Various charts such as bar plots, box plots, and distribution plots are used to examine rating trends, app category performance, pricing patterns, and the relationship between installs, reviews, and ratings. These visual insights help highlight which types of apps perform better and what factors contribute to higher user engagement. Overall, this notebook demonstrates a structured EDA approach and shows how systematic data cleaning and visualization can uncover meaningful insights into app popularity and performance, providing a strong foundation for data-driven decision-making and predictive modeling.

## Time to Minutes conversion function
def convert_to_minutes(time):
    hours = 0
    minutes = 0

    if 'h' in time:
        hours = int(time.split('h')[0])
        time = time.split('h')[1]

    if 'm' in time:
        minutes = int(time.replace('m', '').strip())

    return hours * 60 + minutes
df['Duration_Minutes'] = df['Duration'].apply(convert_to_minutes).astype(int)

    Netflix TV Shows EDA

    Analyzed Netflix content data to compare TV shows and movies, uncovering trends in content type, release patterns, ratings, and country-wise distribution using Python and data analysis techniques

    The project began by importing the Netflix dataset from Kaggle and performing an initial inspection to understand its structure, columns, and data types. Essential Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn were used for data manipulation and visualization. Next, data cleaning was carried out to handle missing values in key columns like director, cast, country, and rating, ensuring the dataset was reliable for analysis. Date-related columns were processed to extract meaningful time-based insights such as year of release. The core analysis focused on comparing TV Shows vs Movies across multiple dimensions. This included analyzing the overall content distribution, trends in content release over the years, popular ratings, and duration patterns. Country-wise and genre-based analysis was also performed to understand Netflix’s global content strategy. Visualizations such as bar charts and count plots were used to clearly present findings, helping identify growth trends, dominant content types, and audience preferences. The project concluded by summarizing insights that highlight how Netflix’s catalog has evolved over time and how TV shows and movies differ in terms of volume, ratings, and release behavior.
